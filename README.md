# بورتفوليو فدوى عبدالرحمن السيف — Full-Stack Web Developer

مبني بنفس لغة التصميم الموصوفة في ملف المواصفات المرفق (خلفية سوداء سينمائية، بالانة كريمية دافئة #DEDBC8/#E1E0CC، خط Almarai + Instrument Serif المائل، حركات framer-motion، أيقونات lucide-react).

## ⚠️ مهم قبل التشغيل

هذا الموقع بُني من الصفر اعتماداً على **ملف مواصفات تصميم** (Prisma) أرسلته، وليس من كود موقعك الفعلي الحالي (لأنه ما توفر). يعني:

- التصميم يطبّق نفس الهوية البصرية الموصوفة (ألوان، خطوط، حركات، أحجام الكروت...) بأمانة.
- لكنه **ليس نسخة طبق الأصل من موقعك الحالي** إذا كان يختلف عن هذه المواصفات في أي تفصيل.
- إذا عندك لقطات شاشة أو رابط مباشر لموقعك الحالي، ابعثهملي وأقارن وأعدّل أي فرق.

## الأقسام المطبَّقة

1. **Hero** — اسمك بحروف عملاقة + وصف + أزرار (Hire Me / View My Projects / Download CV)
2. **About** — نبذة عنك + صورة شخصية (placeholder حالياً) + فقرة تنكشف حرفاً حرفاً عند التمرير
3. **Services** — 10 خدمات بنفس ستايل كروت #212121
4. **Skills** — كل تصنيفات المهارات كـ badges كريمية داخل كارد #101010
5. **Who I Work With** — 5 فئات عملاء بنفس ستايل الكروت
6. **Why Choose Me** — 8 أسباب بأيقونة Check
7. **Projects** — مشروع فوح الكبسة (بغلاف SVG مصمم بنفس الهوية) + مشروعين "Coming Soon"
8. **Contact/Footer** — بريدك وواتسابك

## نواقص تحتاج إجراء منك

| العنصر | الحالة | الإجراء المطلوب |
|---|---|---|
| صورتك الشخصية | placeholder دائري فاضي | ضع صورتك باسم `portrait.jpg` داخل مجلد `public/` |
| غلاف مشروع فوح الكبسة | SVG تصميمي وليس لقطة شاشة حقيقية | استبدل `public/project-foah-cover.svg` بلقطة شاشة حقيقية من موقعك (احفظها بنفس الاسم أو عدّل المسار في `src/components/Projects.tsx`) |
| ملف السيرة الذاتية (CV) | زر "Download CV" يشير لملف غير موجود | ضع ملف PDF باسم `Fadwa-Alsaif-CV.pdf` داخل `public/` |
| مقارنة دقيقة بالتصميم الأصلي | غير ممكنة حالياً | ابعث كود موقعك الحالي أو صور/فيديو له لمطابقة أي تفاصيل مختلفة |

## التشغيل محلياً

```bash
npm install
npm run dev
```

يفتح على `http://localhost:5173` تلقائياً.

## البناء للنشر

```bash
npm run build
```

الناتج في مجلد `dist/` جاهز للرفع على أي استضافة استاتيكية (Vercel, Netlify, GitHub Pages...).
